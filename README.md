### **使用方式**

1. **關閉防毒軟體**，解壓縮密碼為taiwan001

2. 內有2個文件，分別為軟體本身以及**act.key**

3. act.key**請勿移動位置**，否則無法執行

4. 軟體打開後直接**點"永久啟用"**即可

5. 點擊後會**跳出CMD**並詢問是否執行，**請選"是"**

6. 等待一陣子就可以打開設定確認**是否啟用成功**

7. 若啟用成功即**可將CMD和軟體關閉**
